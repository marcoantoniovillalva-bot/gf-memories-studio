# GF Memories Studio — Luxury Photography Website

> Client website for a professional photography studio specialising in pet and family portraits. Built for performance, SEO, and a premium visual experience.

**Live:** [gfmemoriesstudio.com](https://gfmemoriesstudio.com)

---

## Overview

A fast, responsive marketing website for a photography studio. Features a cinematic hero, animated service cards, a retina-sharp lightbox gallery with lazy loading, Instagram integration, cookie consent, and full SEO optimisation with Open Graph and structured data.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 16 (App Router) + React 19 + TypeScript |
| Styling | Tailwind CSS 3.4 |
| Auth | Supabase (SSR) |
| Images | Next.js Image with 2× retina optimisation |
| SEO | Open Graph, Twitter Cards, JSON-LD structured data, sitemap |
| Deployment | Vercel |

---

## Features

- Lightbox gallery with swipe, keyboard navigation, and fullscreen mode
- Retina-optimised image grid (2× quality, proper `sizes` props)
- Expandable service cards and accordion sections
- WhatsApp CTA integration
- Cookie consent banner
- Fully internationalised meta tags (IT/EN)
- Mobile-first responsive layout

## Local Setup

```bash
git clone https://github.com/marcoantoniovillalva-bot/gf-memories-studio
cd gf-memories-studio
npm install
cp .env.local.example .env.local
npm run dev
```
